## 시작하기

### 로컬 개발환경에서 실행하기

1. 이 레포지토리를 클론해 주세요.

   ```bash
   $ git clone https://github.com/kavoom2/ms-choi-popupbook.git
   ```

2. NodeJS v16.0.0 이상을 설치하고, 의존성 패키지들을 설치합니다.

   ```bash
   $ nvm use v16.13.12
   $ npm install
   ```

3. 개발 서버를 실행해 주세요.

   ```bash
   $ npm run start

   // 아래 명령어로 입력하면 WebGL를 디버깅 모드로 실행합니다.
   $ npm run start:webglDebug
   ```
